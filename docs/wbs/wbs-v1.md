# WBS Version 1

Use this draft as the first WBS version in XMind. It intentionally gives the group a broad starting point that can later be validated against the Disjunction Rule and Completion Rule.

```text
iGEO IoMT Project

1. Project Management
   1.1 Project initiation and planning
   1.2 Stakeholder management
   1.3 Risk management
   1.4 Progress monitoring and reporting

2. Requirements Engineering
   2.1 Stakeholder interviews and requirement elicitation
   2.2 Functional requirements for hosting application
   2.3 Requirements for WH Module
   2.4 Requirements for SC Module
   2.5 Security, privacy, and safety requirements
   2.6 Requirements validation and sign-off

3. Supplier and Procurement Management
   3.1 Define supplier selection criteria
   3.2 Evaluate FunIOT and ProfCar
   3.3 Select hardware supplier
   3.4 Select display supplier
   3.5 Contract negotiation and supplier onboarding

4. System Architecture and Design
   4.1 Overall iGEO IoMT architecture
   4.2 Hardware integration design
   4.3 Display and GUI design
   4.4 Hosting application design
   4.5 Module interface design
   4.6 Security and privacy design

5. Implementation
   5.1 iGEO IoMT hardware integration
   5.2 iGEO IoMT display integration
   5.3 Hosting application implementation
   5.4 Hosting application GUI implementation
   5.5 WH Module implementation
   5.6 SC Module implementation
   5.7 Cloud service integration
   5.8 Third-party IoT device integration

6. Testing and Deployment
   6.1 Unit testing
   6.2 Integration testing
   6.3 Security and privacy testing
   6.4 Usability testing with display
   6.5 System acceptance testing
   6.6 Deployment preparation
   6.7 Pilot deployment
   6.8 Final release and handover
```
