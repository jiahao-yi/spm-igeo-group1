## Project Goal

The goal is to prepare one complete English PDF report for the SPM Planning Project. The report should help TOMI Automobiles plan the iGEO IoMT project by covering project definition, stakeholder analysis, supplier decision, WBS, effort estimation, activity network planning, Gantt planning, and resource allocation.

## Repository Structure

```text
docs/                 Main report drafts and planning artifacts
ideas/                Individual member idea inboxes
resources/            Notes about original course resources
final-submission/     Final PDF and ZIP package before submission
```

## Recommended Workflow

1. Each member adds ideas to their own file in `ideas/`.
2. The group discusses and consolidates useful points into `docs/`.
3. Keep diagrams or exports near the related document section.
4. Before final submission, place only the final PDF and required extra files in `final-submission/`.
5. Package the final files as `GROUP_1.zip`.
